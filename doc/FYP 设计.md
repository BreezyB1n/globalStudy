# FYP 设计

名称： GlobalStudy AI (留学项目智能咨询助手)
核心价值： 消除留学信息差，通过 RAG（检索增强生成）技术，为用户提供准确、可溯源的 QS 前 50 名校硕士项目咨询服务。

功能介绍

- 平台抓取QS前50硕士项目介绍与要求，以文件形式存入向量数据库
- 用户通过询问自己感兴趣的项目，AI通过向量数据库获取获取用户感兴趣的项目信息并做介绍

要求：

- 项目尽可能保持架构精简，部署越简单越好，这只是一个毕业设计级别的Demo产品
- 语言使用python + langgraph，其他技术栈我们后续讨论
- 模型使用阿里云百炼 + 硅基流动提供的模型API服务

1. 数据库选择chromaDB 2. Embedding模型使用百炼的text-embedding-v3 3. 前端使用HTML CSS JS三件套 4. 爬虫数据通过Firecrawl API抓取 5. 百炼做主对话 LLM + Embedding 其他暂时先不考虑 6. 部署本地运行即可
